InsetText
=========

### Import
```js
  import InsetText from '@govuk-react/inset-text';
```
<!-- STORY -->

### Usage


Simple
```jsx
<InsetText>Hello</InsetText>
```

Narrow border
```jsx
<InsetText isNarrow>Hello</InsetText>
```
