Input
=====

### Import
```js
  import Input from '@govuk-react/input';
```
<!-- STORY -->

### Usage

Simple
```jsx
<Input type="text" />
```

Custom error colour
```jsx
import { ORANGE } from 'govuk-colours';

<Input type="text" errorColor={ORANGE} error="example" />
```
